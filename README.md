# Georgina McLeod Digital Innovation Website

Static portfolio website for GitHub Pages.

## Files

- `index.html` - main website page
- `style.css` - design and responsive layout
- `script.js` - scroll reveals and gallery lightbox
- `assets/images/` - selected and optimised images
- `assets/images/gallery/` - compressed archive images
- `assets/documents/` - supporting AI strategy/governance documents

## Publish on GitHub Pages

1. Create a new GitHub repository, for example `georgina-digital-innovation`.
2. Upload all files and folders from this package to the repository root.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select `main` and `/root`, then save.
6. GitHub will publish the site at `https://YOUR-USERNAME.github.io/georgina-digital-innovation/`.

## Customise before publishing

Search in `index.html` for:

- `hello@georginamcleod.com` and replace with your preferred email.
- The LinkedIn URL if needed.
- Any project text you want to make more specific.
